# Library Management System

## Description

This is a simple Java console-based Library Management System. 
It allows users to manage books in a library. 
You can add new books, view all books, search for a book by ID, issue books, return books, and check the total number of books. 
It also prevents duplicate book IDs.

## Features

- Add Book
- View all Books
- Search Book by ID
- Issue Book
- Return Book
- Show Total Number of Books
- Prevent duplicate Book IDs
